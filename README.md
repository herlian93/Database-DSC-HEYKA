# Cara Menjalankan Notebook di Github
Notebook dapat langsung dibaca melalui GitHub tanpa perlu menjalankannya.
1. Buka file berikut pada repository
   [Notebook_HEYKA](Notebook_Asuransi_Lengkap.ipynb)
2. Github akan menampilkan:
   - Kode Python
   - Output Hasil Analisis
   - Grafik dan Visualisasi
   
# Cara Menjalankan Notebook di Google Collab
1. Buka website [Google.Collab](https://colab.research.google.com)
2. Pilih menu upload notebook
3. Upload file berikut: [Notebook_HEYKA](Notebook_Asuransi_Lengkap.ipynb)
4. Setelah terbuka, jalankan setiap sel dengan tombol "Run" secara urut dari atas atau menggunakan tombol "Run All"

# Cara Menjalankan Notebook di VS Code
1. Pastikan telah terinstal:
   - Python versi terbaru
   - Visual Studio Code
   - Extension Python
   - Extension Jupiter
2. Buka terminal di VS Code lalu jalankan:
   *pip install pandas numpy matplotlib seaborn scikit-learn xgboost*
3. Download file berikut: [Notebook_HEYKA](Notebook_Asuransi_Lengkap.ipynb)
4. Klik open file pada VS Code. Pilih file yang sudah didownload 
6. Klik "Run" dan "Start Debuging"
