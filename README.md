# Cara Menjalankan Notebook di Github
Notebook dapat langsung dibaca melalui GitHub tanpa perlu menjalankannya.
1. Buka file berikut pada repository
   [Notebook_HEYKA]
2. Github akan menampilkan:
   - Kode Python
   - Output Hasil Analisis
   - Grafik dan Visualisasi
